1. What is an if statement?
    - An if statement is a conditional statement used to do something only if the condition is True.

2. Difference between if and if...else and if...elif...else.
    - if: used to do something if condition is met(True).
    if...else: used to do something if condition is True and something else if it is False.
    if...elif...else: used when we have multiple conditions to check before doing something finally(else).

3. Comparison operators
==
!=
<
>
<=
>=
    - == : returns True if both elements(LHS and RHS) are the same.
    != : Exact opposite of == (returns True if both elements(LHS and RHS) are not the same).
    < : returns True if element value on LHS is smaller than element value on RHS.
    > : returns True if element value on LHS is greater than element value on RHS.
    <= : returns True if element value on LHS is smaller than or equal to element value on RHS.
    >= : returns True if element value on LHS is greater than or equal to element value on RHS.

4. Logical operators
and
or
not
    - and : returns True if both statements(LHS and RHS) are True.
    or : returns True if either of LHS or RHS or both statements are True.
    not: used to invert the truth value of a Boolean expression or object(unary).

5. Nested if statements
    - A nested if statement means placing an if statement inside another if statement. This allows you to check for a second condition only if the first condition turns out to be true.

6. When should we use elif instead of multiple if statements?
    - Use elif when your conditions are mutually exclusive, meaning only one choice can be true at a time.
    Use multiple if statements when you want to check independent conditions, meaning more than one action can happen.

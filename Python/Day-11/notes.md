1. What is Object-Oriented Programming? Why do we use it?
    - Object-Oriented Programming (OOP) is a software design paradigm that structures code around "objects" rather than logic and functions. We use it to build highly modular, reusable, and scalable applications by modeling real-world entities, making complex systems much easier to maintain and collaborate on.

3. What is the purpose of __init__()? When is it called?
    - The purpose of __init__() is to initialize an object's attributes with specific values when the object is created. It acts as the constructor in Python, setting up the initial state of the object so it is ready to use.

4. What is self? Why is it required inside methods?
    - The self keyword represents the specific instance of the class that is currently executing the code. It acts as a reference variable, allowing an object to access its own attributes and methods within the class definition.

5. What is the difference between:

* Attribute
* Method

Give an example of each.
    - An attribute is a variable that stores data about an object, while a method is a function that defines the actions the object can perform.
    Attribute Example: my_car.color = "Red" (stores the car's color).
    Method Example: my_car.start_engine() (runs the code to turn the car on).

6. Why is OOP useful for large applications? Give one real-world example.
    - OOP is useful for large applications because it breaks massive codebases into independent, manageable modules, allowing hundreds of developers to work on separate components simultaneously without breaking each other's code.
